# Troubleshooting Guide

## Common Issues and Solutions

### Wallet Connection Issues

#### Wallet Won't Connect to Stellar Tip Jar

**Symptoms:**
- "Allow" button doesn't work
- No connection dialog appears
- Wallet extension doesn't respond

**Solutions:**
1. **Ensure Freighter is Properly Installed**
   - Check browser extensions menu
   - Look for Freighter in extensions list
   - Disable and re-enable it

2. **Unlock Your Wallet**
   - Click Freighter icon in toolbar
   - If it shows "Unlock" - enter password
   - Refresh Stellar Tip Jar page

3. **Refresh the Page**
   - Use Ctrl+R or Cmd+R
   - Or click the refresh button
   - Try connecting again

4. **Clear Browser Cache**
   - Go to browser settings
   - Find "Clear browsing data"
   - Select "Cookies and cached images"
   - Restart browser

5. **Try Incognito/Private Mode**
   - Open new incognito window
   - Test if it works there
   - If yes, your extensions might have issues

6. **Try Different Browser**
   - Test on Chrome, Firefox, or Edge
   - Narrows down the issue
   - Might be extension conflict

7. **Reinstall Freighter**
   - Remove Freighter from extensions
   - Wait 1 minute
   - Reinstall from freighter.app
   - Create new wallet or import from recovery phrase

#### Freighter Extension Crashed

**Symptoms:**
- Extension disappeared
- Won't load at all

**Solutions:**
1. Check extensions menu - enable if disabled
2. Restart browser completely
3. If still not working, reinstall it
4. Go to freighter.app for fresh download

### Transaction Issues

#### Transaction Won't Send

**Symptoms:**
- Send button does nothing
- No wallet pop-up appears

**Solutions:**
1. **Check Your Balance**
   - Open Freighter
   - View your account balance
   - Ensure you have enough XLM
   - Need: Tip amount + Fee + 1 XLM minimum

2. **Check Internet Connection**
   - Try loading another website
   - If slow, may be connection issue
   - Try wired connection if possible

3. **Wallet Locked?**
   - Click Freighter icon
   - If locked, enter password
   - Refresh Tip Jar page

4. **Try Again**
   - Close any popups
   - Look for "Send" button again
   - Click once (wait 2 seconds)
   - Check if wallet pops up

#### "Insufficient Balance" Error

**Symptoms:**
- Error when trying to send
- "Not enough XLM" message

**Solutions:**
1. **Add More XLM**
   - Freighter shows your balance
   - Buy more XLM from an exchange
   - Withdraw to your wallet address
   - Wait for confirmation (5-10 seconds)

2. **Send Smaller Amount**
   - Reduce tip amount
   - Account for fees
   - You need Tip + Fee + 1 XLM minimum

3. **Check Fee Amount**
   - Review total before sending
   - Fees should be small (under 0.1 XLM)
   - If larger, something's wrong

4. **Withdraw from Another Wallet**
   - If you have another Freighter account
   - Send XLM to this wallet
   - Then use it for tips

#### Transaction Stuck Pending

**Symptoms:**
- "Pending" status for 2+ minutes
- No success or error message
- Amount still reserved in wallet

**Solutions:**
1. **Wait a Bit Longer**
   - Stellar is fast (5-10 seconds usually)
   - Sometimes takes up to 1 minute
   - Very rarely takes 2+ minutes

2. **Refresh the Page**
   - Page might not have updated
   - Check if transaction now shows complete
   - Refresh wallet too

3. **Check Transaction Status**
   - Open Freighter
   - Look at transaction history
   - See if it shows as confirmed

4. **Try a New Transaction**
   - If truly stuck (2+ minutes)
   - Try sending again to same/different creator
   - Your balance may have temporarily locked the funds

5. **Contact Support**
   - If problem persists
   - Provide transaction details
   - Include timestamp
   - Include error messages if any

#### "Invalid Address" Error

**Symptoms:**
- Error mentions "invalid address"
- Can't send to this creator

**Solutions:**
1. **Refresh the Page**
   - Creator address may not have loaded
   - Refresh and try again
   - Cache issue might be cause

2. **Try Different Creator**
   - Test with another creator
   - If that works, first creator has issue
   - Report to support

3. **Check Your Wallet**
   - Make sure wallet is up to date
   - Update extensions if needed
   - Try unlocking/locking wallet

4. **Contact Support**
   - If persists with other creators
   - Might be platform bug
   - Include which creators fail

### Profile and Account Issues

#### My Profile Won't Save

**Symptoms:**
- "Save" button doesn't work
- Loses changes after refresh
- Error message (if any)

**Solutions:**
1. **Check Required Fields**
   - All required fields filled?
   - Mark * usually indicates required
   - Profile picture uploaded?
   - Username chosen?

2. **Try Smaller Changes**
   - Edit one field at a time
   - Save after each change
   - Pinpoint which field causes issue

3. **Image File Size**
   - Reduce image size (compress)
   - Keep under 5MB typically
   - Use JPG or PNG format
   - Try image compressor website

4. **Clear Cache and Try Again**
   - Clear browser cache
   - Refresh Stellar Tip Jar
   - Try editing profile again
   - Wait for save to complete before refreshing

5. **Try Different Browser**
   - Use different browser
   - See if saves there
   - If yes, extension conflict likely

#### Can't Find My Profile Link

**Symptoms:**
- Don't know my profile URL
- Need to share but can't find link

**Solutions:**
1. **Find It in Your Account**
   - Sign in to Stellar Tip Jar
   - Go to your profile page
   - Check browser address bar
   - That's your profile link!

2. **Format:**
   - Usually: `https://tipjar.com/yourname`
   - Copy from browser address bar
   - This is your shareable link

3. **Test the Link**
   - Open in new incognito window
   - Make sure it's publicly accessible
   - Check it displays correctly

#### Profile Isn't Public

**Symptoms:**
- Profile 404 or not found
- Can't see my own profile

**Solutions:**
1. **Check Privacy Settings**
   - Go to profile settings
   - Look for "Visibility" or "Private/Public"
   - Ensure set to "Public"
   - Save if you changed it

2. **Verify Profile is Completed**
   - Profile might need all fields filled
   - Complete your profile setup
   - Check for incomplete sections

3. **Test After Saving**
   - Wait a few seconds
   - Open profile in incognito window
   - Should now be visible

### Tip History and Viewing Issues

#### Can't See My Tips

**Symptoms:**
- Sent tip but it doesn't show up
- Can't find where to view tips

**Solutions:**

**For Supporters:**
1. **Check Creator's Profile**
   - Go back to creator
   - Refresh page
   - Look for tip history
   - Your tip might appear there

2. **Check Your Wallet**
   - Open Freighter
   - Look at transaction history
   - Your tip should show as sent
   - If shows there, message delayed updating

3. **Wait a Moment**
   - Page might not update immediately
   - Wait 30 seconds
   - Refresh page
   - Tip should appear

**For Creators:**
1. **Check Your Dashboard**
   - Sign in
   - Go to your dashboard
   - Look for "Recent Tips" section
   - Refresh if not showing

2. **Check Tip History**
   - Click "View All Tips" or similar
   - See chronological list
   - Verify amount and timestamp

3. **Verify Deposit**
   - Go to your wallet
   - Check transaction history
   - Tip funds should appear

#### Creator's Tip History is Empty

**Symptoms:**
- No tips showing even though received many
- History showing 0 tips

**Solutions:**
1. **Refresh Page**
   - Page might not have loaded
   - Refresh or reload completely
   - Clear cache if needed

2. **Check Privacy Settings**
   - Profile hidden? Make public
   - Tips might not display when private
   - Verify visibility setting

3. **Withdraw and Check**
   - Go to withdraw page
   - Should show your total
   - If shows amount, tips received just not displaying
   - Might be display bug

4. **Contact Support**
   - If tips withdrawn suggest funds received
   - But no history shows
   - Report bug to support

### Withdrawal Issues

#### Can't Withdraw Tips

**Symptoms:**
- "Withdraw" button inactive
- No withdrawal option visible
- Getting error when trying

**Solutions:**
1. **Check You Have Tips!**
   - Dashboard should show balance
   - If $0 or 0 XLM, nothing to withdraw
   - First receive a tip

2. **Check Minimum Balance**
   - Stellar needs 1 XLM minimum
   - After withdrawal, 1 XLM must remain
   - Can't withdraw everything

3. **Wallet Connected?**
   - Ensure wallet is connected
   - Sign in if needed
   - Check wallet is unlocked

4. **Try After Waiting**
   - Recent tips might unlock after time
   - Wait 5 minutes and try again
   - Check if cooldown exists

#### Withdrawal Took Too Long

**Symptoms:**
- Clicked withdraw 10+ minutes ago
- Still shows pending
- No XLM in wallet

**Solutions:**
1. **Wait a Bit More**
   - Stellar takes 5-20 seconds usually
   - Might take up to 1 minute on busy days
   - Very rarely takes longer

2. **Check Transaction Status**
   - Open Freighter
   - Look at transaction history
   - Should show withdrawal attempt

3. **Refresh Pages**
   - Refresh both pages
   - Check if now shows complete
   - Wallet might show it already received

4. **Check Wallet Address**
   - Went to correct wallet?
   - Might have sent to wrong address
   - Verify address in transaction

5. **Contact Support if Truly Stuck**
   - After 5+ minutes of pending
   - Provide transaction hash if visible
   - Include tip total and timestamp

### Search and Discovery Issues

#### Can't Find a Creator

**Symptoms:**
- Creator not in search results
- Profile doesn't exist

**Solutions:**
1. **Check Spelling**
   - Search is usually case-insensitive
   - But spelling must match exactly
   - Try different variations
   - Numbers vs letters (0 vs O)

2. **Use Exact Username**
   - Don't search "john" if username is "johndoe"
   - Try just "john" to see suggestions
   - Platform might suggest similar usernames

3. **Direct Link Method**
   - If you know their username: `tipjar.com/username`
   - Try typing directly in address bar
   - Sometimes easier than searching

4. **Ask Creator Directly**
   - Message them on social media
   - Ask for their Stellar Tip Jar link
   - They'll appreciate the support intent!

5. **Creator Might Not Have Profile**
   - Not all creators set up profiles
   - Creator set to private
   - Creator deleted profile
   - Contact them to set one up

#### Search Not Working

**Symptoms:**
- Search bar doesn't respond
- Results don't appear
- Getting search errors

**Solutions:**
1. **Refresh Page**
   - Might be temporary glitch
   - Refresh and try again

2. **Clear Cache**
   - Clear browser cache
   - Clear cookies
   - Refresh site

3. **Try Direct Link**
   - Instead of searching
   - Type `tipjar.com/username` directly
   - Bypass search completely

4. **Contact Support**
   - If search is broken
   - Limit your activities
   - Report issue with details

### Browser and Device Issues

#### Website Looks Broken

**Symptoms:**
- Layout is messed up
- Text is unreadable
- Buttons don't work

**Solutions:**
1. **Hard Refresh**
   - Ctrl+Shift+R (Cmd+Shift+R on Mac)
   - Forces reload of all resources
   - Clears cached CSS/images

2. **Clear Browser Cache**
   - Settings → Clear browsing data
   - Select "All time"
   - Include cookies and cache
   - Refresh site

3. **Try Different Browser**
   - Chrome, Firefox, Safari, Edge
   - See if happens in other browsers
   - Narrows down issue

4. **Disable Extensions**
   - Too many extensions can cause issues
   - Disable all except Freighter
   - Test site
   - Re-enable gradually to find culprit

#### Site Super Slow

**Symptoms:**
- Loads very slowly
- Buttons take long to respond
- General lag

**Solutions:**
1. **Check Internet Connection**
   - Run speed test
   - Restart router if slow
   - Try wired connection

2. **Clear Cache**
   - Cached files might be corrupted
   - Clear all browsing data
   - Fresh load should be faster

3. **Close Other Tabs**
   - Too many tabs use memory
   - Close unnecessary tabs
   - Browser runs faster

4. **Restart Browser**
   - Close completely
   - Wait 10 seconds
   - Reopen and test

### Mobile / App Issues

#### Mobile Website Doesn't Work

**Symptoms:**
- Buttons too small on phone
- Layout broken on mobile
- Can't send tips from phone

**Solutions:**
1. **Zoom Out** (if text too large)
   - Pinch to zoom out
   - Or use browser zoom controls
   - Find comfortable viewing size

2. **Try Desktop Mode**
   - Some mobile browsers have "Desktop Mode"
   - Request desktop version
   - Might work better

3. **Mobile Wallet**
   - Freighter may not work on mobile
   - Try other wallets (xBull, Lobstr have apps)
   - Download mobile wallet app

4. **Desktop for Important Actions**
   - Easier to send tips on desktop
   - Mobile viewing OK
   - Use desktop for withdrawals
