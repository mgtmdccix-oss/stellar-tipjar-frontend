# Creator Profile Setup

This guide walks you through creating and customizing your creator profile on Stellar Tip Jar.

## Getting Started

Before you set up your profile, make sure you:
- Have Freighter wallet installed
- Have signed in to Stellar Tip Jar
- Know your desired username

If you need help with these prerequisites, see [GETTING_STARTED.md](GETTING_STARTED.md).

---

## Creating Your Profile

### Step 1: Access Profile Settings

1. **Sign In**
   - Go to Stellar Tip Jar
   - Click "Sign In"
   - Connect your wallet
   - Approve in Freighter

2. **Navigate to Settings**
   - Click your account menu (usually top-right)
   - Look for "Profile" or "Settings"
   - Click "Edit Profile" or "Creator Profile"

3. **Start Profile Creation**
   - You'll see a form with various fields
   - Fill them out as you go through this guide

### Step 2: Choose Your Username

Your username is your profile's web address and identity.

**Good Username Characteristics:**
- Easy to spell and remember
- Relevant to your content
- Consistent with other platforms
- Not too long (under 30 characters)
- Lowercase letters and numbers recommended

**Examples:**
- `artistjohn` (Clear and simple)
- `musicproducer99` (Descriptive)
- `devstudio` (Professional)

**Avoid:**
- Numbers that look like letters (l, 1, O, 0)
- Special characters (if possible)
- Names already taken (you'll get an error)
- Very long strings

**Your Profile URL:**
- Your username becomes: `tipjar.com/yourname`
- Can't be changed after creation (usually)
- Test the username before submitting

### Step 3: Upload Profile Picture

Your profile picture is how people identify you.

**Best Practices:**
- **Use a clear headshot** (for personal creators)
- **Use your logo** (for brands/studios)
- **Professional quality** - good lighting, clear focus
- **Simple background** - avoid distracting elements

**Technical Requirements:**
- Format: JPG, PNG, WebP
- Recommended size: 400x400px or larger
- Maximum file size: Check the form
- Square aspect ratio works best
- Under 5MB typically

**How to Upload:**
1. Click "Upload Photo" or icon
2. Select image from your computer
3. Crop/adjust if needed
4. Click "Save" or "Confirm"

**Profile Picture Ideas:**
- Professional headshot
- Your brand/business logo
- Avatar or illustration
- Beautiful product photo
- Stylized photo with personality

### Step 4: Write Your Bio

Your bio introduces supporters to you and your work.

**Bio Guidelines:**
- Length: 100-300 characters (varies by platform)
- Be clear about what you create
- Show personality
- Make it interesting

**Bio Structure:**
1. What you do (job/role)
2. What makes you unique
3. Call to action (optional)

**Example Bios:**

*Artist:* "Digital artist creating custom fantasy illustrations. Every commission gets my full creative vision ✨"

*Musician:* "Independent electronic artist. Creating ambient and experimental music in my home studio."

*Developer:* "Open-source software developer. Building tools to make the web faster and more accessible."

*Writer:* "Fiction writer and storyteller. TBD: [current project]. Supporter tips enable more writing time!"

*Content Creator:* "YouTuber creating indie game reviews and gaming culture analysis. Your tips help fund production!"

**Bio Tips:**
- Use emojis to add personality
- Be honest about what you do
- Don't oversell yourself
- Make it readable
- Avoid all caps

### Step 5: Add Your Links

Connect supporters to your other platforms.

**Useful Links to Include:**
- Website or portfolio
- Twitter / Social media
- YouTube or streaming platform
- GitHub (if developer)
- Instagram, TikTok, Discord
- Patreon, Ko-fi, or other platforms

**How to Add Links:**
1. Look for "Social Links" or "Links" section
2. Click "Add Link" or similar
3. Choose the platform (if option available)
4. Paste your full URL
5. Click "Add" or "Save"

**Link Format Tips:**
- Use full URLs: `https://twitter.com/yourname`
- Not just usernames
- Verify link is correct before saving
- Remove old/broken links
- Keep them current

**Recommended Link Order:**
1. Your main website
2. Most active social platform
3. Content platform (YouTube, etc.)
4. Other relevant platforms

### Step 6: Review Your Profile

1. **Check Everything**
   - Username looks correct
   - Picture displays well
   - Bio reads nicely
   - Links are accurate
   - Privacy is set correctly

2. **View as Others See It**
   - Click "Preview" if available
   - Open your profile in incognito window
   - See how it looks to visitors

3. **Make Adjustments**
   - Edit any fields that need changes
   - Re-upload picture if needed
   - Fix typos
   - Reorder links

### Step 7: Make Your Profile Public

1. **Check Privacy Settings**
   - Look for "Visibility" or "Private/Public"
   - Ensure set to "Public" (unless you want it private initially)

2. **Save Your Profile**
   - Click "Save" or "Publish"
   - Wait for confirmation
   - Check that it saved (you'll see success message)

3. **Your Profile is Live!**
   - Supporters can now find you
   - Share your profile link
   - Tips can start coming in

## Customizing Your Profile

Once your basic profile is set up, consider these customizations:

### Add a Banner/Cover Image

Some platforms let you add a banner image:
- Upload a wider image (aspect ratio ~3:1)
- Use consistent branding
- Make it eye-catching
- Include your name or logo

### Write a Longer Bio

If the initial bio is limited, many platforms have:
- Extended profile description
- About me section
- Portfolio blurbs

Use this to:
- Tell your story
- Explain your creative process
- Share what you're passionate about
- Give more context

### Add Categories/Tags

If available, add tags like:
- Your art style (Abstract, Digital, Photography)
- Your industry (Tech, Art, Music)
- Your platform (YouTube, Twitch, etc.)
- Your niche (Indie, Educational, etc.)

This helps supporters find you!

### Set Up Notifications

Configure how you're notified:
- Email notifications for tips
- Push notifications (if app)
- Frequency (immediate, daily digest)
- Which events to notify about

## Best Practices for Profile Growth

### Profile Tips for Attracting Supporters

✅ **Do:**
- Be authentic and genuine
- Show your personality
- Keep information current
- Use high-quality images
- Link to active platforms
- Update your bio regularly

❌ **Don't:**
- Use blurry or low-quality photos
- Leave sections blank
- Include outdated information

## Troubleshooting

### Can't Find Edit Profile Button

**Solution:**
- Ensure you're logged in
- Click on your profile name/account menu
- Look for "Settings," "Profile," or "Edit"
- Try Account → Preferences

### Profile Won't Save

**Possible causes:**
- Required field is empty
- File too large
- Connection issue

**Solution:**
- Check all required fields are filled
- Reduce image file size
- Try refreshing page
- Clear browser cache
- Try different browser

### Picture Not Uploading

**Solution:**
- Check file format (JPG, PNG, WebP)
- Reduce file size (compress image)
- Check file size limit
- Try different browser
- Check internet connection

### Username Already Taken

**Solution:**
- Choose a different username
- Add numbers or underscores
- Combine name parts differently
- Consider using your full name

### Profile Not Showing Tips

**Solution:**
- Ensure profile is set to "Public"
- Check tip system is enabled
- Verify profile saved correctly
- Share your profile link

### Can't Change Username

**Note:** Most platforms don't allow username changes after creation.

**Solution:**
- Create a new profile with new username
- Use "Display Name" field to change visible name
- Contact support for exceptional circumstances
