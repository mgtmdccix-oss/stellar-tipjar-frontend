# Security Best Practices

Your security on Stellar Tip Jar is crucial. This guide explains how to protect your cryptocurrency and personal information.

## Overview

Stellar Tip Jar uses blockchain technology which is inherently secure, but your personal security habits matter. The biggest risks usually come from:
- Weak passwords
- Sharing recovery phrases
- Public WiFi
- Phishing attacks
- Unverified links

This guide helps you avoid these.

---

## Wallet Security

Your wallet is your most valuable asset on this platform. Protect it carefully.

### Recovery Phrase Protection

**What is Your Recovery Phrase?**
- 12-24 words that create your wallet
- Can recreate your wallet from scratch
- Gives access to all your funds
- Cannot be reset or changed

**Protecting Your Recovery Phrase:**

✅ **Do:**
- Write it down on paper (not digital)
- Store in a safe location (safe, safety deposit box)
- Keep multiple copies in different locations
- Store away from your computer
- Use a secure notebook with a lock

❌ **Never:**
- Store digitally (photos, notes, emails)
- Shared with anyone (even support staff)
- Posted online or in screenshots
- Stored in password managers (usually)
- Discussed in public

**Recovery Phrase Best Practices:**
1. Write it immediately when creating wallet
2. Store first copy in safe place
3. Consider second copy in different location (safety deposit box)
4. Test recovery (optional): Write phrase down, import to new wallet, verify

### Password Security

**Creating a Strong Password:**

Your wallet password protects your wallet from local access.

**Strong Password Characteristics:**
- 12+ characters (longer is better)
- Mix uppercase and lowercase
- Include numbers
- Include special characters (!@#$%^&*)
- Not a dictionary word
- Not personal info (name, birthday, pet names)
- Unique (don't reuse passwords)

**Examples of Weak Passwords:**
- `password123` (dictionary word + numbers)
- `John1985!` (personal information)
- `12345678` (all numbers)
- `abc123` (too short)

**Examples of Strong Passwords:**
- `Tr0p!cal*Sunset#2024` (mixed case, numbers, symbols)
- `BluePanda$Crypto&Moon!` (memorable but complex)
- `K@tingScout1994Purple!` (personal but complex)

**Password Manager:**
- Use a password manager (Bitwarden, 1Password, LastPass)
- Generates and stores strong passwords
- Reduces need to remember many passwords
- Must have strong master password

### Using Your Wallet Safely

**On Your Personal Device:**
- Use only on your own computer
- Keep your device secure
- Don't let others use your computer
- Use device password protection
- Enable fingerprint/face unlock if available

**On Shared Devices:**
1. Avoid using wallet on shared computers
2. If you must:
   - Log out completely when done
   - Clear browser cache and cookies
   - Don't save passwords
   - Delete browser history

**Password Managers:**
- Consider using one for your wallet password
- Adds encryption layer of protection
- Makes strong passwords manageable
- Backup/store master password securely

### Wallet Unlocking

**Keeping Wallet Unlocked:**
- Lock wallet when not in use
- Some browsers auto-lock after inactivity
- Phishing can steal funds from unlocked wallets
- Don't leave wallet unlocked unattended

**Lock Your Wallet:**
1. Click Freighter icon
2. Click "Lock" or "Sign Out"
3. Must re-enter password to use wallet
4. Takes 2 seconds, adds security

## Transaction Safety

Before sending any transaction, verify everything.

### Before Every Transaction

**Checklist Before Clicking "Send":**

- [ ] Are you on the correct website? (Check URL)
- [ ] Is the recipient address correct? (Check first/last characters)
- [ ] Is the amount correct? (Verify amount shown)
- [ ] Have you read this address before? (Is it familiar?)
- [ ] Are you on a secure network? (Not public WiFi)
- [ ] Is your device secure? (Antivirus up to date?)
- [ ] Did you use a verified link? (From creator, not elsewhere)

### Double-Check Recipient

**Verify Creator Address:**
1. Click on creator's profile
2. Verify name matches who you want to tip
3. Check username looks familiar
4. Look at their tip history (does it exist?)
5. See recent tips (is it active?)

**Don't Tip:**
- Suspicious/new profiles with no history
- If anything feels off
- Links from random messages
- Unverified creators

### Amount Verification

**Before Confirming Payment:**
1. Check the amount you entered
2. Verify in the wallet pop-up
3. Understand the fees (usually small)
4. Total cost = Tip + Fees
5. Ensure you have enough balance

**If Amount Seems High:**
- You might have forgotten XLM decimals
- Check the fee percentage
- Verify total cost
- Don't confirm if unsure, contact support

### Test Transactions

**Large Amounts Strategy:**
1. Send small test tip first ($1-5 equivalent)
2. Verify it goes through correctly
3. Verify creator receives it
4. Only then send larger amounts

**New Creator:**
1. Start with small tip ($1-2 equivalent)
2. Ensure transaction works
3. Build comfort with their address
4. Increase tip size if desired

## Account Security

### Strong Account Security

**For Stellar Tip Jar Account:**

1. **Use Unique Passwords**
   - Different password for site account (if applicable)
   - Different from wallet password
   - Different from email password
   - Use password manager to manage

2. **Two-Factor Authentication (2FA)**
   - If Stellar Tip Jar offers 2FA, enable it
   - Use authenticator app (Google Authenticator, Authy)
   - Not SMS 2FA if possible (less secure)
   - Save backup codes in safe place

3. **Email Account Security**
   - Secure email with strong password
   - Enable 2FA on email account
   - Email is recovery method for most accounts
   - Protect it fiercely

4. **Log Out on Shared Devices**
   - Always log out completely
   - Don't stay signed in
   - Clear cookies after session
   - Restart browser to be sure

### Recognizing Phishing

**Phishing: Fake websites that steal your info**

**Phishing Red Flags:**
- Unexpected emails asking to "verify" account
- Links from random people about crypto
- Urgent warnings about account action
- Requests for password or recovery phrase
- Slight misspellings in email address
- Generic greetings ("Dear Customer")
- Links don't match apparent sender

**Example Phishing Email:**
```
From: help@stellar-tipjar.com [WRONG - real domain is tipjar.com]
Subject: URGENT: Verify Your Account Now!

Dear User,

Your account has been compromised! Click here to verify:
[link to fake website that looks like Stellar Tip Jar]

Never reply with passwords or recovery phrases!
```

**Phishing Prevention:**
1. **Verify Links**
   - Hover over links (don't click yet)
   - Check if URL matches what you expect
   - Slight misspellings are red flag
   - When unsure, type URL yourself

2. **Verify Senders**
   - Check sender email address carefully
   - Hover over sender name
   - Verify it matches official domain
   - Be suspicious of any crypto emails

3. **Check Website URL**
   - Always click to address bar first
   - Verify correct URL before signing in
   - Look for `https://` and lock icon
   - Bookmark pages to avoid typos

4. **Legitimate Sites Never Ask For:**
   - Recovery phrases
   - Private keys
   - Passwords
   - Personal ID numbers
   - Payment info (unless buying)

5. **When in Doubt:**
   - Don't click links in emails
   - Go to website directly in address bar
   - Contact support through official channels
   - Never panic and rush

### Email Security

**Protecting Your Email:**
- Strong, unique password
- 2FA enabled
- Review connected apps/accounts
- Check login activity regularly
- Update recovery phone number

**Your email is critical because:**
- Used to reset passwords
- Receives security warnings
- Might receive withdrawal confirmations
- Allows account recovery

Protect it like your wallet!

## Network Security

### Public WiFi Safety

**Risks of Public WiFi:**
- Attackers can intercept traffic
- Someone can see what you're doing
- Malware can be installed
- Passwords can be stolen

**Avoid on Public WiFi:**
- Signing in to accounts
- Sending cryptocurrency
- Accessing sensitive information
- Anything with passwords

**If You Must Use Public WiFi:**
1. **Use VPN**
   - Virtual Private Network encrypts traffic
   - Use reputable VPN (ProtonVPN, ExpressVPN, Mullvad)
   - Encrypts all traffic
   - Costs money but worth it

2. **Turn Off Auto-Connect**
   - Don't auto-connect to WiFi
   - Choose networks manually
   - Disable Bluetooth when possible
   - Disable NFC if available

3. **Use Mobile Data Instead**
   - Use phone hotspot if possible
   - Mobile data more secure than public WiFi
   - Don't do this on 4G with data limits

### Home Network Safety

**Your Home Network:**
- Change default WiFi password
- Use WPA3 encryption (or WPA2)
- Change default router password
- Enable router firewall
- Keep router firmware updated

**Device Security:**
- Use device firewall (built-in usually)
- Keep operating system updated
- Run antivirus software
- Don't use outdated browsers
- Disable unnecessary services

## Scam Prevention

### Common Crypto Scams

**1. Fake Support/Recovery Services**
- Someone messages: "I can help recover your funds"
- They ask for recovery phrase or money upfront
- It's a scam - you lose everything
-Official support won't ask for recovery phrases

**2. Giveaway Scams**
- "Send 1 XLM, receive 10 back!"
- It's not real
- Your XLM disappears
- Never send expecting return

**3. Impersonation Scams**
- Someone pretending to be support team
- Asks for password or recovery phrase
- Steals your wallet
- Official support never asks for these

**4. Investment Scams**
- Promise high returns (50%, 100%+)
- Usually yield nothing
- Your money disappears
- If too good to be true, it is

### Protecting Against Scams

**Do:**
- ✅ Verify who you're talking to
- ✅ Be skeptical of unsolicited offers
- ✅ Research before investing
- ✅ Use official support channels
- ✅ Ask friends/community if unsure
- ✅ Trust your gut if something feels off

**Don't:**
- ❌ Send money to earn money
- ❌ Share recovery phrases with anyone
- ❌ Trust unsolicited messages
- ❌ Follow links from random people
- ❌ Assume crypto is anonymous (it's traceable)
- ❌ Panic and act quickly

### Red Flags

Someone messaging you about your crypto is usually red flag:
- "I heard you lost your funds..."
- "Invest with me for guaranteed returns"
- "Your wallet has been compromised"
- "Act now or lose access"
- "Too good to be true" offers

If anyone messages you:
1. Don't engage
2. Don't provide information
3. Don't send money
4. Report them if possible
5. Block them

## Operational Security (OpSec)

"Operational Security" means protecting yourself through habits.

### Internet Hygiene

**Browser Extensions:**
- Only install trusted extensions
- Freighter from official freighter.app
- Minimal extensions (avoid clutter)
- Update regularly
- Review permissions

**Browser Hygiene:**
- Disable cache for wallets (optional, advanced)
- Use different browser for crypto
- Don't browse suspicious sites on crypto browser
- Regular updates

**Anti-Malware:**
- Use quality antivirus (Windows Defender + Malwarebytes)
- Run full scans monthly
- Keep definitions updated
- Be cautious of downloads

### Device Security

**Keep Device Secure:**
- Screen lock (password, fingerprint, face)
- Full disk encryption (built-in usually)
- Don't lend devices to others
- Antivirus installed
- Firewall enabled

**Physical Security:**
- Don't leave device unattended
- Close laptop when stepping away
- Lock door when working with crypto
- Webcam cover optional but ok

### Backup Security

**Back Up Your Recovery Phrase:**
- Written copy in safe location
- Second copy in different location
- Encrypted copy (optional, advanced)
- Never all digital
- Test recovery once (optional)

**Never Back Up**
- Recovery phrase digitally
- Private keys anywhere
- Screenshots of wallet
- Passwords on sticky notes

## Incident Response

### If You Realize a Problem

**Your Wallet Was Compromised:**
1. Stop transactions immediately
2. Don't enter recovery phrase online
3. Document what happened
4. Contact Stellar Tip Jar support
5. Move funds to new wallet if possible
6. Change all related passwords (email, exchanges)

**You Entered Recovery Phrase Online:**
1. Create new wallet immediately
2. Move all funds to new wallet
3. Abandon old wallet
4. Consider that phrase compromised permanently
5. Document and learn from mistake

**You Sent Money by Mistake:**
1. Blockchain is permanent - money can't be reversed
2. Try contacting recipient (if you know them)
3. Ask them to return it (unlikely)
4. Report scam to platform if applicable
5. Contact Stellar Tip Jar support
6. Accept lesson learned for future

**You Lost Your Recovery Phrase:**
1. If still have wallet access:
   - Keep using it
   - Keep password secure
   - Export recovery phrase if possible (some allow)
   - Don't move funds without it

2. If lost access completely:
   - Funds are likely gone
   - Might contact support (won't help)
   - Learn lesson: always save phrase!

### Documentation for Support

If contacting support about security issue:
- Don't share recovery phrase or private keys
- Share transaction IDs if applicable
- Describe what happened clearly
- Include timestamps
- Include which wallet address if relevant
- Be honest about what you did
